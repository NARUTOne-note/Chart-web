## Web 动画

- 增强动画与页面元素之间的关联性
- 不要为了动画而动画，要有目的性
- 动画不要过于缓慢，否则会阻碍交互
- 结合产品及业务的创意交互动画
- 考虑动画的性价比

**12原则**:

- 挤压和拉伸：物体存在质量且运动时质量保持不变的概念
- 预备动作：运动不倾向于突然发生
- 演出布局：确保对象在场景中得以聚焦，让场景中的其它对象和视觉在主动画发生的地方让位
- 连续运动和姿态对应：连续运动是绘制动画的每一帧，姿态对应是通常由一个 assistant 在定义一系列关键帧后填充间隔
- 跟随和重叠动作：意味着让过渡或动画的效果以不同速度来运行
- 缓入缓出：在一个动画过程中计时功能是一种描述变化速率的方式
- 弧线运动：遵循「缓入缓出」的时候它们很少沿直线运动——它们倾向于沿弧线运动
- 次要动作：主动画正在发生，次要动作可以增强它的效果
- 时间节奏：调整时间节奏可以帮动画的内容和交互方式变得更出众
- 夸张手法：最常用来为某些动作刻画吸引力和增加戏剧性的
- 扎实的描绘：当动画对象在三维中应该加倍注意确保它们遵循透视原则
- 吸引力：注重细节和动作相结合而打造吸引性的结果
